# First Brief - Visual Comfort / Art Direction Lead

**Role:** CHAT-VISUAL-001  
**Status:** Waiting for Game Director task assignment

## Problem To Solve

Watch Officer needs a visual direction that is comfortable for perception and credible for a maritime decision simulator.

Current prototype visuals are functional, but not final.

The game must not become:

- cartoon;
- toy-like;
- engineering-only;
- harsh chart/CAD interface;
- noisy arcade UI.

## Desired Feel

The player should feel:

- calm;
- focused;
- oriented;
- professionally guided;
- not overloaded;
- not visually punished.

## Draft Visual Principles

- Water should be soft, dark maritime blue-green, with subtle depth.
- Safe water should be legible but not neon.
- Shallow/danger should be warm and visible but not aggressive by default.
- Lines should be broad enough for comfort.
- Marks should be large and recognizable.
- AIS/vector cues should be smooth, clear, and not needle-thin.
- Warnings should escalate visually without panic.
- HUD should feel like a calm watch officer display, not a flight cockpit.
