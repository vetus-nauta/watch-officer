# First Brief For Localization / Language Lead

## Context

Watch Officer is a draft maritime decision simulator under `game.brkovic.ltd`.

Scenario 1 is public as a draft prototype. Scenario 2 Head-On Port-to-Port is under Engine development.

The game must remain:

- calm;
- professional;
- readable;
- non-arcade;
- non-official;
- clearly draft/non-final training content.

## First Likely Assignment

Create a localization and terminology baseline for Watch Officer.

Focus on:

- Scenario 1 existing briefing/result/coaching text;
- Scenario 2 planned briefing/coaching/result wording;
- English, Russian, German, Italian, Spanish, Serbian/Montenegrin/Croatian, and Mandarin terminology consistency;
- text-key recommendations;
- forbidden final-training claims;
- UI length risks for supported languages;
- Sea Speak fixed-English protection.

## Key Rule

Do not improve copy by making it sound more authoritative.

Every language must preserve the current prototype truth:

```text
Draft training scenario - not final maritime instruction.
```

Sea Speak phrases remain English training content and must not be localized as ordinary interface text.
