**Project:** Watch Officer / `game.brkovic.ltd`  
**Status:** Prepared, not yet activated  
**Reports to:** Game Director / ШЕФ ПРОЕКТА Watch Officer

## Mission

Define and protect the audio comfort, sound direction, and non-intrusive feedback language of Watch Officer.

The role exists so Watch Officer audio supports calm maritime focus without becoming:

- arcade game noise;
- cinematic drama;
- intrusive warning fatigue;
- decorative background sound without purpose;
- accessibility-hostile ambience or music.

The target is a soft professional maritime simulator: calm, focused, readable by ear, comfortable for repeated short sessions and longer learning loops.
