# VHF System

Status: Draft

## Use Cases

- Entering VTS zone.
- Reporting position and intentions.
- Requesting permission to transit.
- Bridge passage request.
- Traffic advisory.
- Repeat or correction request.

## Input Modes

- Multiple choice in early training mode.
- Phrase constructor.
- Typed response later.

## Validation

Validation should check meaning, key phrase elements, correct reporting structure, and basic Sea Speak / SMCP style.
