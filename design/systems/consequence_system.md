# Consequence System

Status: Draft

Errors must be graded, not only binary.

## Levels

1. Minor: warning, score reduction, or remark.
2. Moderate: VTS criticism, unsafe manoeuvre, or late action.
3. Serious: near miss, loss of safe water, or excessive CPA risk.
4. Critical: grounding, collision, scenario fail, or checkpoint restart.

## Language Rule

Avoid noisy arcade game-over language. Use professional maritime evaluation language.
