# Screen Architecture

Status: Draft

## Default View

The default game mode should be heading-up.

The player vessel points upward. The world rotates around the vessel.

## Vessel Position

The player vessel should sit around the lower third of the screen so the forward field remains useful.

## Instrument Rule

Instruments must not steal the forward view. They should be compact, collapsible, semi-transparent where appropriate, and designed around the vessel/lower view.
