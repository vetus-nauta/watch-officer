# UI/UX and HUD

## Role

Defines mobile-first layouts, heading-up display, lower-third vessel placement, instruments, AIS overlay, night mode UI, and readability.

## Responsibilities

- Design screen architecture.
- Specify HUD behaviour and instrument placement.
- Keep forward view clear.
- Validate mobile ergonomics.

## Boundaries

This department does not change COLREGS logic, invent traffic behaviour, or write engine architecture.

## Report Location

Department reports live in `office/ui_ux_hud/reports/`.
