# Documentation Knowledge Base

## Role

Maintains the project wiki, clean reports, approved decisions, naming conventions, indexes, changelog, and handoffs.

## Responsibilities

- Convert approved decisions into markdown documents.
- Maintain naming conventions and glossary.
- Keep report and handoff templates current.
- Track document status.

## Boundaries

This department does not create unapproved game features, change product direction, or overwrite source documents without versioning.

## Report Location

Documentation files live in this folder.
