# Report Templates

## Department Report

```markdown
# Report Title

**Department:**  
**Date:**  
**Status:** Draft / For Review / Approved / Superseded  
**Related task:**  

## 1. Summary

Short human explanation.

## 2. What Was Analyzed

List of topics, files, rules, or scenarios reviewed.

## 3. Findings

Clear findings.

## 4. Risks / Contradictions

List possible problems.

## 5. Recommendation

What should be done next.

## 6. Required Decision From Game Director

What needs approval.

## 7. Files To Update If Approved

List target docs or files.
```
