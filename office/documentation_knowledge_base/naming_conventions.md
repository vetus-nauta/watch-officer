# Naming Conventions

## Project Name

Use `Watch Officer` for the product unless the Game Director approves another final title.

## Repository Folders

Use lowercase snake_case for documentation folders and files.

## Terms

Preferred:

- traffic target
- encounter vessel
- AI traffic vessel
- scenario vessel
- vessel target

Avoid in formal documentation:

- random bot
- arcade enemy
- game over

## Status Labels

- Draft
- For Review
- Approved
- Superseded
