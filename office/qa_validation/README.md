# QA and Validation

## Role

Tests scenario logic, finds contradictions, verifies MAMS/IALA interpretation, verifies COLREGS situation logic, and checks UI readability.

## Responsibilities

- Review rule drafts for contradictions.
- Mark uncertain educational content.
- Validate scenarios against intended learning outcomes.
- Track scenario test reports and bug lists.

## Boundaries

This department does not implement fixes directly or invent mechanics without a report.

## Report Location

QA reports live in `office/qa_validation/reports/` and scenario tests live in `office/qa_validation/scenario_test_reports/`.
