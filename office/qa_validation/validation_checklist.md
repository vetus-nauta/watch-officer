# Validation Checklist

Status: Draft

## Maritime Rules

- [ ] IALA/MAMS rules verified.
- [ ] COLREGS abstraction reviewed.
- [ ] VHF/VTS prompts reviewed.
- [ ] Uncertain rules marked as requiring verification.

## Scenario Logic

- [ ] Objective is clear.
- [ ] Fail states are fair.
- [ ] Evaluation language is professional.
- [ ] Scenario does not teach unsafe behaviour.

## UI Readability

- [ ] Forward view remains clear.
- [ ] Vessel placement supports heading-up navigation.
- [ ] AIS labels are readable.
- [ ] Mobile layout is usable.
