# Scenario Design Rules

Status: Draft

## Scenario Target

- 3-7 minutes.
- 1-3 key tasks.
- 1-5 navigation decisions.
- 1-2 pressure elements.
- Clear professional evaluation.

## Each Scenario Must Define

- Environment.
- Time mode.
- Weather.
- Vessel type.
- Navigation elements.
- Traffic targets.
- Communication events.
- Objectives.
- Checkpoints.
- Fail states.
- Evaluation criteria.
