# Art and Visual Direction

## Role

Defines visual style, vessel icons, buoy readability, day/night atmosphere, fog and visibility style, color palette, and icon language.

## Responsibilities

- Keep the visual language calm, serious, professional, and maritime.
- Ensure marks and vessels are readable at game scale.
- Provide asset lists and reference notes.

## Boundaries

This department does not change scenario mechanics, define COLREGS rules, or write core code.

## Report Location

Department reports live in `office/art_visual_direction/reports/`.
