# Engine and Godot

## Role

Defines Godot project structure, scenes, movement code, camera behaviour, data loading, collision detection, scenario runtime, and export strategy.

## Responsibilities

- Prepare implementation plans.
- Define prototype architecture.
- Identify technical risks.
- Keep implementation aligned with approved product documents.

## Boundaries

This department does not alter the game concept, rewrite maritime rules, or decide art direction.

## Report Location

Department reports live in `office/engine_godot/reports/`.
