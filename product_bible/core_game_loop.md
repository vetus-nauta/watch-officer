# Core Game Loop

1. Select a short scenario.
2. Read the immediate maritime environment.
3. Control vessel speed and heading.
4. Interpret marks, safe water, traffic, and AIS cues.
5. Respond to one or more decision points.
6. Handle any VHF/VTS prompt.
7. Complete, fail, or restart from checkpoint.
8. Receive a professional evaluation report.

## Scenario Unit

Each scenario should define:

- Environment.
- Time mode.
- Weather.
- Vessel type.
- Navigation elements.
- Traffic targets.
- Communication events.
- Objectives.
- Checkpoints.
- Fail states.
- Evaluation criteria.

Target scenario structure:

- 3-7 minutes.
- 1-3 key tasks.
- 1-5 navigation decisions.
- 1-2 pressure elements.
- Clear professional evaluation.
